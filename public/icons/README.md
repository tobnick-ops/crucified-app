# Icons für PWA

## Icon-Anforderungen

Die App benötigt Icons in verschiedenen Größen für PWA-Installation:

- **72x72** - Klein
- **96x96** - Klein
- **128x128** - Mittel
- **144x144** - Mittel
- **152x152** - Mittel
- **192x192** - Groß (empfohlen)
- **384x384** - Groß
- **512x512** - Groß (empfohlen)
- **180x180** - Apple Touch Icon

## Icon-Generator

1. Öffne `icon-generator.html` im Browser
2. Die Icons werden automatisch generiert
3. Klicke auf "Download" für jedes Icon
4. Speichere die Icons im `public/icons/` Verzeichnis

## Erstelle eigene Icons

### Empfohlene Tools

- **Figma**: Professionelles Design-Tool
- **Canva**: Einfaches Design-Tool
- **Adobe Illustrator**: Vektorbasiertes Design
- **GIMP**: Kostenloses Bildbearbeitungs-Tool

### Design-Richtlinien

- **Farbe**: Tempel-Ästhetik (Gold: #d4af37, Blau: #1e3a5f, Weiß)
- **Theme**: Tempel, Kreuz, Glauben
- **Stil**: Minimalistisch, klar erkennbar
- **Format**: PNG (transparenter Hintergrund)

### Icon-Inspiration

- Tempel-Architektur
- Kreuz
- Buch (Bibel)
- Licht (Kerze)
- Kombination aus Tempel + Kreuz

## Aktuelle Icons

Die aktuellen Icons sind Platzhalter. Ersetze sie durch professionelle Icons für Production.

## Installation

1. Erstelle oder lade Icons herunter
2. Benenne sie entsprechend (z.B. `icon-192x192.png`)
3. Speichere sie im `public/icons/` Verzeichnis
4. Die App lädt sie automatisch über `manifest.json`
